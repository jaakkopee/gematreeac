gematreeac input

Notice that the session and DeepMem databases are visible to all and only session memory can be erased!

Click on commands or enter them in the textfield below:

CLEAR empties session memory

SHOW

shows classified words in current session

ALPHABET

prints alphabet in use

ADDTODEEPMEM

adds all words in current session to DeepMem (permanent memory, not erasable)

SHOWDEEPMEM

lists all words and gematria values in DeepMem

Enter a number to search words from current session and DeepMem by gematria value

To add data to the session memory, submit any text (word, sentence etc.) written or copy-pasted in the text field below
